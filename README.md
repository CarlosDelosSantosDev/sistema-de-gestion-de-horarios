# Sistema de Gestión y Planeación de Horarios Institucionales 

Solución de software modular diseñada para automatizar y agilizar el complejo proceso de asignación de periodos, materias y docentes dentro del *Departamento de Idiomas y Comunicación Continua* de la Universidad Politécnica de Gómez Palacio. Desarrollado e implementado con éxito durante la Estancia Profesional II.

###  Estado del Proyecto: 100% Funcional e Implementado
> **Nota de alcance:** El sistema fue adoptado formalmente por el departamento administrativo de la institución, reduciendo drásticamente las horas invertidas en planeación manual y eliminando los conflictos de empalmes de horarios.

###  Características Principales

* **Interfaz Interactiva Drag & Drop:** Implementé un panel visual dinámico que permite a los coordinadores asignar profesores y materias a sus respectivos bloques de horario simplemente arrastrando y soltando elementos en tiempo real.
* **Algoritmo de Validación por Periodo:** El sistema evalúa la disponibilidad del docente y las aulas según el periodo académico seleccionado, previniendo errores organizacionales.
* **Exportación y Formateo Oficial:** Automatización total de la salida documental, estructurando las tablas de horarios en el formato oficial institucional listo para la firma de las autoridades correspondientes.

###  Tecnologías Utilizadas

* **Lenguaje de Programación:** C# (.NET)
* **Base de Datos:** Estructurada para el manejo dinámico de periodos, materias, salones y docentes.
* **Componentes Visuales:** Lógica avanzada de manipulación de eventos de interfaz (Drag & Drop de Windows Presentation Foundation o WinForms).
